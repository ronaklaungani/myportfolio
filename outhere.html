The Idea

My Client, Mohit wanted to build a product that could help users locate and assist their family and friends with just a tap. It is a permission based system that enables the user to allow/disallow a family member/friend to see their location. Moreover, it allows the user to take actions like call, book a cab, draft a whatsapp text, find nearby places, and more.”

Conclusions (Functional Requirements)

On logging in, the users can sync their contacts
Users can send/receive permission requests to/from the contacts who are already using OutHere
For others, the user can send a join OutHere request link via SMS, email or WhatsApp
Once a user sends a permission request to a contact, the contact is added to their list as "pending". The user can still cannot view the location of the contact
Likewise, when a user recieves a permission reuqests, it sits under the "Permissions" tab for the user to review and accept/reject
Once accepted, the contact can then view the user's location
The user can remove the contact from the permissions list any time.
The user can also enter in hibernate mode to deny (not remove) location permission to all accepted contacts
While viewing the location, the user can perform actions like make a call, draft a whatsapp message, book a taxi and explore places around the contact.

Initial Ideas for the interface design

1 - The user can sign-up with Facebook or Google+ to enable quick log-in.

2 - My Friends Screen - has the list of all the contacts that can view user's location.

3 - A switch to activate Hibernate Mode<.

4 - An easy access button to add new friends or groups.

5 - Permissions Screen - displays the list of all the permission requests for the user to respond.

6 - The Map Interface - displays variety of data including the name, picture, location, battery status & distance from the user.

7 - A map layout visually shows the location of the contact.

8 - The bottom part of the screen shows all the actions for the user like - make a call, open in whatsapp, book a taxi, etc.

9 - On clicking "Show Places" or "Show Nearby Places" the user can find places around the contact like a petrol pump, bakery, restaurant, Police stations, etc. This data is fetched from Google.

10 - The app also shows a detailed page about the selected near by place with data fetched from Google.

Designing the Pin

Since it is the most important element of the app, me and Mohit spent some time on designing just the right pin. Of-course it was Mohit to decide upon the final design but was really fun to redesign such a small element. It needed to stand out from the interface but also follow the scheme I had adopted. It should also look really great on a map, which again was custom to match the feel of the app.



The Hibernate Button

This was the most difficult part of this project - to figure out how to tell “You’re hidden” to someone with just one icon.
The usual icons available for such kind of messages didn’t really give across the true meaning of the “hibernate” functionality. We needed something new and I wanted it to be fun so users don’t only associate it with something really serious and feel little less guilty about it.
So, I decided to take advantage of the personas that we were already using. It carried the theme forward and kinda added a personal touch for the user. I came up with 2 options. First one was really interesting and is my favourite too. It is much more quirky and does portray the sense of “hiding”.


Option 1: Hibernate Mode OFF

Option 1: Hibernate Mode ON



Why the Second Option?

On tapping the character(chosen by the user during app launch) the character hides behind the floating card. It gives a sense of the character hiding behind a sheet to avoid being seen by anyone - which is what the hibernate functionality does. But the development team suggested it would be difficult for them to pull this sheet off (pun intended), with the limited time in hand. Hence, I came up with a second option. Instead of pulling the card up like a sheet, the character hides behind the card when the hibernate mode is turned ON. The eye movement adds a quirky element to the functionality.

Near-by Places

A part of the map interface functionality is to allow the users to find places around the located friend. Example - Gas Station, Bars, Hospitals, Ports, etc. We designed custom badges for these places to be placed on the map.
